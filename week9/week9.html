Browser Object Model (BOM): is a collection of properties and methods about the browser and computer screen. There is no official standard; however, several of the methods are supported by all major browsers.

Global variables are created using const, let, or var. Global variables can be accessed in all parts of the program. Global Variables are part of the window object.

Dialogs: alert(), confirm(), prompt() are not part of the ECMAScript standard although all major browsers support them as methods of the window object. window.alert() method will pause the execution of the program and display a message in the dialog box. window.confirm() dispays a confirmation dialog with an argument to press OK or Cancel. wondow.prompt() will display a message with an input field to unter text.

window.open() will open a new window. close() method can be used to close a window. to get screen information you can use window.screen. postfixes for that include: height, width, availHeight, availWidth, colorDept...

Cookies: Cookies are small files saved locally on a user's computer. Cookies store information that can be retrieved between requests. Cookies can only be read by a web page from the same domain that set them. They are limited to 4kb of data and 20 cookies per domain. These allow for personalizing a users expierence.

HTML5 contains several APIs that help gain access to hardware such as cameras, batteries, geolocation, and graphics card.
