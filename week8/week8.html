The translate(x,y) function moves an element x from the left, and y from the top: (transform: translate(45px, -45px);) Transform does require vendor prefixing for IE9 and iOS8. You can also use translateX or translateY if only wanting to move vertically or horizontally. You can use scale to scale an object up or down. (transform: scale(1.5, 0.25);) scale(1) would leave an element the same size. (transform: rotate(10deg);) Rotate will adjust the rotation. (transform: skew(15deg, 4deg);) will skew perspective.

Transitions are a way to animate objects with css rather than using javascript. You can use a transition-duration to set how long the animation goes.
Animations will still need prefix's.
