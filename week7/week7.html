Notes:

I can see the functionality of memoization (.cache) property. This could be great if working with a program where you call a function multiple times and need to debug the program. You would be able to find the issue in the program.

Async functions allow you to write asynchronous code as if it was synchronous. This is done by utilizing the 'await' operator before the function. Using this and the promise feature allows us to write code without having to nest all the different functions to work together.

It seems that best practice would be to stay as close to functional programming as possible.
